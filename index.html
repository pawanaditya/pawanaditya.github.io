Face Mask Detection Algorithm

The implementation uses MATLAB and the Viola-Jones algorithm to detect facial features like the mouth, nose, and eyes. Covid 19 Analysis Using R programming

The COVID-19 analysis project aimed to assess the impact of age and gender on COVID-19 fatality rates using a dataset from Johns Hopkins University. The analysis was conducted using R, and the dataset contained information on cases, deaths, and recoveries. Key findings showed that older individuals (average age of 68.5 years for deceased) were more likely to die compared to younger people (average age of 48.8 years for survivors). Additionally, the analysis revealed that males had a higher fatality rate (8.4%) compared to females (3.6%). The overall death rate from COVID-19 in the dataset was 5.8%. The project provided insights into the demographic factors influencing COVID-19 mortality.
